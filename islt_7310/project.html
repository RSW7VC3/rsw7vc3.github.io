SERS Mobile Application (Silent Emergency Response System) CAPSTONE PROJECT

Project Description:

This project I worked on the application team that worked with a web team to create the Silent Emergency Response System (SERS) Application. This is a conceptual project designed to tackle the issue of having to call the police when a non-verbal communication method does not exist. The app team created an application with 4 unique features. Including a tip line, a map of the University of Missouri and emergency communication systems on campus, a profile to store user data, and the ability to chat with the emergency dispatch in case of a crisis. On this project, I helped with I the creation and security of the database and the creation of the pages of the application.

Competencies: Google Firebase DB, XCode/Swift, Teamwork, and presentation skills/public speaking skills
